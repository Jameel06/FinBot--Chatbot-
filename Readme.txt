# 💰 FinBot – AI Finance Assistant

FinBot is an AI-powered finance chatbot built using **Streamlit + Groq API + Yahoo Finance API**.
It is designed to provide intelligent, real-time, and professional responses to finance-related queries.

---

## 🚀 What This Project Does

FinBot can help users with:

* 📊 Mutual Funds & SIP explanations
* 📈 Stock market queries
* 💰 Budgeting & personal finance
* 🧾 Basic tax-related guidance
* 🔴 Live stock price updates (via Yahoo Finance API)

Unlike basic chatbots, this is **NOT rule-based** — it uses an **AI model (LLM)** to generate smart responses.

---

## 🧠 Key Features

* 🤖 AI-powered chatbot using Groq API
* 📡 Real-time stock data integration (Yahoo Finance)
* 💬 Context-aware conversation
* 🎨 Clean UI built with Streamlit
* 🔐 Secure API handling using environment variables

## ⚙️ How to Run This Project (IMPORTANT)

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

---

### 2️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

---

### 3️⃣ Add API Keys (VERY IMPORTANT)

This project WILL NOT RUN without API keys.

You need to create a `.env` file in the root folder and add:
GROQ_API_KEY=your_groq_api_key_here


### 🔐 Why `.env` is used?

* To keep API keys **secure**
* Prevents exposing sensitive data on GitHub
* This is **industry standard practice**

### 4️⃣ Run the App
streamlit run app.py
